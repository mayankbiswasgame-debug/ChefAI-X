# 🍳 ChefAI X

ChefAI X is an AI-powered recipe generator that helps users instantly create recipes for different dishes. Users can type a dish name, use voice input, choose a language, adjust serving size, and get a realistic recipe with ingredients, instructions, cooking time, nutrition details, tips, and even video tutorial links.

The project is designed with a clean and mobile-friendly interface, making it easy to use on both desktop and smartphone screens.

## 📄 File Description

### `index.html`

Landing page with the ChefAI X title and start button.

### `app.html`

Main application page where users can search recipes, use voice input, choose language, and interact with recipe features.

### `style.css`

Contains all styles including layout, colors, buttons, loading animation, chat design, and responsive UI.

### `script.js`

Contains all logic such as:

* API request handling
* Voice recognition
* Speech output
* Search history
* Save/share/download features
* Language support
* Loading animation
* Recipe formatting

## ⚙️ How To Run Locally

1. Download or clone this repository
2. Open the project folder in VS Code
3. Add your Groq API key inside `script.js`
4. Open `index.html` in your browser

Example:

```javascript
const API_KEY = "YOUR_GROQ_API_KEY_HERE";
```
